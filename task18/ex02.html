Câu hỏi 1: HTML là gì?

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu dùng để tạo cấu trúc cho trang web.

Câu hỏi 2: CSS có tác dụng gì?

CSS (Cascading Style Sheets) dùng để định dạng và trang trí giao diện cho trang web.

Câu hỏi 3: JavaScript làm được gì?

JavaScript là ngôn ngữ lập trình giúp trang web tương tác và xử lý logic phía client.
